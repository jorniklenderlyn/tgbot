# Задача 4.6.1 — ExponentialBackoffRetry *(дочерняя к 4.6)*

## Основная информация

| Поле | Значение |
|------|----------|
| ID | 4.6.1 |
| Эпик | E4 — Конвейер генерации ответов |
| Родительская задача | **4.6** MessageQueue |
| Статус | Бэклог |
| Зависит от | — (утилита, не зависит от других задач) |
| Влияет на | **4.4** (LLMClient использует для retry), **4.6** (MessageQueue использует для расчёта `next_retry_at`) |
| Тип связи с эпиком | Влияет на E4 |

## Описание

Реализовать утилиту `with_retry` в файле `utils/retry.py` — декоратор с exponential backoff, применяемый в нескольких местах системы: `LLMClient` (задача 4.4) при вызовах к API, `MessageQueue` (задача 4.6) при вычислении следующей попытки.

Exponential backoff — стандартный паттерн устойчивости: при сбое ждём 1 сек, при следующем — 2 сек, затем 4, 8, 16... Это снижает нагрузку на сервис при восстановлении и предотвращает thundering herd (шквал запросов от всех клиентов одновременно).

Jitter (±20% случайного разброса) добавляется к каждой задержке, чтобы несколько экземпляров системы не делали retry строго одновременно.

## Что нужно разработать

- Функцию `compute_delay(attempt: int, base: float, max_delay: float) -> float` — возвращает задержку с jitter
- Декоратор `with_retry(max_attempts, base_delay, max_delay, exceptions)` — применяется к sync и async функциям
- Логирование каждой попытки с номером и задержкой

## Критерий приёмки

- Декоратор применим к обычным и `async` функциям
- Задержки с base=1: 1s → 2s → 4s → 8s → 16s (каждая ±20% jitter)
- `max_delay` ограничивает максимальную задержку (по умолчанию 30 сек)
- Параметр `exceptions` — tuple исключений, при которых делать retry (остальные пробрасываются сразу)
- При исчерпании попыток — пробрасывается исходное исключение (не `RetryError`-обёртка)
- Каждая попытка логируется: `attempt N/M failed: <error>. Retry in X.Xs`

## Примерное решение

```python
# utils/retry.py
import asyncio
import functools
import logging
import random
import time
from typing import Callable, TypeVar

logger = logging.getLogger(__name__)
F = TypeVar("F", bound=Callable)


def compute_delay(attempt: int, base: float = 1.0, max_delay: float = 30.0) -> float:
    """Exponential backoff с ±20% jitter."""
    delay = min(base * (2 ** (attempt - 1)), max_delay)
    jitter = delay * 0.2 * (random.random() * 2 - 1)  # [-20%, +20%]
    return max(0.0, delay + jitter)


def with_retry(
    max_attempts: int = 5,
    base_delay: float = 1.0,
    max_delay: float = 30.0,
    exceptions: tuple = (Exception,),
):
    """Декоратор: retry с exponential backoff для sync и async функций."""
    def decorator(fn: F) -> F:
        if asyncio.iscoroutinefunction(fn):
            @functools.wraps(fn)
            async def async_wrapper(*args, **kwargs):
                last_error = None
                for attempt in range(1, max_attempts + 1):
                    try:
                        return await fn(*args, **kwargs)
                    except exceptions as e:
                        last_error = e
                        if attempt == max_attempts:
                            break
                        delay = compute_delay(attempt, base_delay, max_delay)
                        logger.warning(
                            f"{fn.__name__}: attempt {attempt}/{max_attempts} "
                            f"failed: {e}. Retry in {delay:.1f}s"
                        )
                        await asyncio.sleep(delay)
                raise last_error
            return async_wrapper  # type: ignore
        else:
            @functools.wraps(fn)
            def sync_wrapper(*args, **kwargs):
                last_error = None
                for attempt in range(1, max_attempts + 1):
                    try:
                        return fn(*args, **kwargs)
                    except exceptions as e:
                        last_error = e
                        if attempt == max_attempts:
                            break
                        delay = compute_delay(attempt, base_delay, max_delay)
                        logger.warning(
                            f"{fn.__name__}: attempt {attempt}/{max_attempts} "
                            f"failed: {e}. Retry in {delay:.1f}s"
                        )
                        time.sleep(delay)
                raise last_error
            return sync_wrapper  # type: ignore
    return decorator


# Использование:
# @with_retry(max_attempts=5, base_delay=1.0, exceptions=(APIError, APIConnectionError))
# def complete(self, prompt: str) -> LLMResponse:
#     ...
```
