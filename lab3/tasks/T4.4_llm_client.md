# Задача 4.4 — LLMClient: вызов LLM с retry/backoff

## Основная информация

| Поле | Значение |
|------|----------|
| ID | 4.4 |
| Эпик | E4 — Конвейер генерации ответов |
| Статус | Бэклог |
| Зависит от | **4.3** (PromptBuilder → готовый промпт) |
| Влияет на | **4.5** (DraftManager получает текст черновика), **4.6** (MessageQueue перехватывает при сбое) |
| Тип связи с эпиком | Влияет на E4 |

## Описание

Реализовать `LLMClient` в файле `pipeline/llm_client.py` — обёртку над OpenAI-совместимым клиентом (используется OpenRouter). Принимает готовый промпт из `PromptBuilder` (задача 4.3) и возвращает текст черновика.

При сбоях сети или ошибках `5xx` от API — автоматический retry с exponential backoff (задержки: 1 сек → 2 → 4 → 8 → 16, max 5 попыток). Если все попытки исчерпаны — бросает `LLMUnavailableError`, что сигнализирует `MessageQueue` (задача 4.6) поставить сообщение в очередь.

При деградации API (latency > 10 сек) — вызывает callback `on_slow()`, через который Aiogram-бот уведомит владельца об ожидании.

**Важно:** в API уходит только анонимизированный промпт из `PromptBuilder` — исходный текст переписки туда не попадает.

## Что нужно разработать

- Датакласс `LLMResponse(text: str, latency_ms: float, model: str)`
- Исключение `LLMUnavailableError`
- Класс `LLMClient` с методом `complete(prompt: str, model: str) -> LLMResponse`
- Конфигурация через `.env`: `OPENROUTER_API_KEY`, `LLM_MODEL`, `LLM_SLOW_THRESHOLD_MS`

## Критерий приёмки

- `complete()` возвращает `LLMResponse` с текстом черновика
- При сбое API — retry до 5 раз с задержками 1, 2, 4, 8, 16 сек
- После 5 неудачных попыток — `raise LLMUnavailableError`
- Latency логируется в каждом вызове
- При `latency_ms > 10_000` — вызывается `on_slow()` callback
- API-ключ читается из переменной окружения, не хардкодится

## Примерное решение

```python
# pipeline/llm_client.py
import os
import time
import logging
from dataclasses import dataclass
from typing import Callable
from openai import OpenAI, APIError, APIConnectionError

logger = logging.getLogger(__name__)


@dataclass
class LLMResponse:
    text: str
    latency_ms: float
    model: str


class LLMUnavailableError(Exception):
    pass


class LLMClient:
    def __init__(
        self,
        api_key: str | None = None,
        base_url: str = "https://openrouter.ai/api/v1",
        slow_threshold_ms: float = 10_000,
        on_slow: Callable | None = None,
    ):
        self._client = OpenAI(
            api_key=api_key or os.environ["OPENROUTER_API_KEY"],
            base_url=base_url,
        )
        self._slow_threshold = slow_threshold_ms
        self._on_slow = on_slow

    def complete(
        self,
        prompt: str,
        model: str | None = None,
    ) -> LLMResponse:
        model = model or os.getenv("LLM_MODEL", "openai/gpt-4o-mini")
        max_retries = 5
        delay = 1.0
        last_error: Exception | None = None

        for attempt in range(1, max_retries + 1):
            try:
                t0 = time.monotonic()
                response = self._client.chat.completions.create(
                    model=model,
                    messages=[{"role": "user", "content": prompt}],
                    max_tokens=300,
                    temperature=0.7,
                )
                latency_ms = (time.monotonic() - t0) * 1000

                if latency_ms > self._slow_threshold and self._on_slow:
                    logger.warning(f"LLM slow: {latency_ms:.0f}ms")
                    self._on_slow()

                return LLMResponse(
                    text=response.choices[0].message.content.strip(),
                    latency_ms=latency_ms,
                    model=model,
                )
            except (APIError, APIConnectionError) as e:
                last_error = e
                logger.warning(
                    f"LLM attempt {attempt}/{max_retries} failed: {e}. "
                    f"Retry in {delay:.0f}s"
                )
                time.sleep(delay)
                delay = min(delay * 2, 30)

        raise LLMUnavailableError(
            f"LLM недоступен после {max_retries} попыток: {last_error}"
        )
```
