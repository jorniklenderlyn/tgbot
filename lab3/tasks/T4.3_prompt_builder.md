# Задача 4.3 — PromptBuilder: сборка промпта

## Основная информация

| Поле | Значение |
|------|----------|
| ID | 4.3 |
| Эпик | E4 — Конвейер генерации ответов |
| Статус | Бэклог |
| Зависит от | **4.1** (ClassificationResult), **4.2** (FewShotRetriever → примеры) |
| Влияет на | **4.4** (LLMClient получает готовый промпт) |
| Тип связи с эпиком | Влияет на E4 |

## Описание

Реализовать `PromptBuilder` в файле `pipeline/prompt_builder.py`. Собирает финальный промпт для LLM из четырёх блоков:

1. **Persona** — системная инструкция из `persona.json` (E2): кто такой пользователь, его стиль, типичные обороты.
2. **Few-shot примеры** — пары «входящее → ответ» из Qdrant (задача 4.2), показывают LLM образцы стиля.
3. **Контекст диалога** — последние N сообщений переписки с конкретным собеседником.
4. **Воспоминания** — факты из Mem0 (E3) о собеседнике и пользователе, релевантные текущему сообщению.

Главное ограничение: промпт должен укладываться в `max_tokens`. Если не влезает — обрезать few-shot примеры (убирать самые старые по одному), затем контекст диалога. Persona и воспоминания не обрезаются — они важнее.

Если `ClassificationResult.is_sensitive=True`, добавить в промпт явную инструкцию: «Не бери на себя финансовых или временных обязательств».

## Что нужно разработать

- Класс `PromptBuilder` с методом `build(...) -> str`
- Шаблон промпта (строка с плейсхолдерами)
- Логика обрезки по токенам через `tiktoken`

## Критерий приёмки

- Метод `build(persona, examples, context, memories, classification, max_tokens=6000) -> str`
- Промпт всегда содержит блок persona и инструкцию отвечать в стиле пользователя
- Few-shot примеры форматируются как `Входящее: ...\nОтвет: ...`
- Блок воспоминаний добавляется только если `len(memories) > 0`
- При `is_sensitive=True` — добавляется предупреждение в промпт
- Итоговый промпт не превышает `max_tokens` (проверяется через `tiktoken`)
- Если после обрезки примеров промпт всё равно > лимита — контекст обрезается до последних 5 сообщений

## Примерное решение

```python
# pipeline/prompt_builder.py
import tiktoken
from pipeline.classifier import ClassificationResult

ENCODING = tiktoken.get_encoding("cl100k_base")

TEMPLATE = """\
Ты — {name}. {style_summary}
Типичные обороты: {phrases}.

{sensitive_warning}

## Примеры твоих ответов
{fewshot_block}

## Что ты знаешь о собеседнике и себе
{memory_block}

## Текущий диалог
{context_block}

Напиши ответ одним сообщением в своём стиле, без пояснений:\
"""

SENSITIVE_WARNING = (
    "ВАЖНО: сообщение затрагивает чувствительную тему. "
    "Не бери на себя финансовых или временных обязательств.\n"
)


class PromptBuilder:
    def build(
        self,
        persona: dict,
        examples: list[dict],
        context: list[dict],
        memories: list[str],
        classification: ClassificationResult,
        max_tokens: int = 6000,
    ) -> str:
        fewshot_lines = [
            f"Входящее: {ex['incoming']}\nОтвет: {ex['reply']}"
            for ex in examples
        ]
        memory_block = (
            "\n".join(f"- {m}" for m in memories)
            if memories else "(нет данных)"
        )
        sensitive_warning = SENSITIVE_WARNING if classification.is_sensitive else ""

        # Обрезаем few-shot пока не влезем в лимит
        while True:
            context_msgs = context[-10:]  # последние 10 сообщений
            context_block = "\n".join(
                f"{'←' if m['direction'] == 'incoming' else '→'} {m['text']}"
                for m in context_msgs
            )
            prompt = TEMPLATE.format(
                name=persona.get("name", "пользователь"),
                style_summary=persona.get("style_summary", ""),
                phrases=", ".join(persona.get("phrases", [])[:5]),
                sensitive_warning=sensitive_warning,
                fewshot_block="\n\n".join(fewshot_lines) or "(нет примеров)",
                memory_block=memory_block,
                context_block=context_block,
            )
            if len(ENCODING.encode(prompt)) <= max_tokens:
                return prompt
            if fewshot_lines:
                fewshot_lines.pop(0)  # убираем самый старый пример
            elif len(context) > 5:
                context = context[-5:]  # обрезаем контекст
            else:
                return prompt  # больше нечего обрезать
```
