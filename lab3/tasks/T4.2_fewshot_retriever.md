# Задача 4.2 — FewShotRetriever: поиск примеров в Qdrant

## Основная информация

| Поле | Значение |
|------|----------|
| ID | 4.2 |
| Эпик | E4 — Конвейер генерации ответов |
| Статус | Бэклог |
| Зависит от | **E1** (few-shot индекс должен быть уже загружен в Qdrant) |
| Влияет на | **4.3** (PromptBuilder использует найденные примеры) |
| Тип связи с эпиком | Влияет на E4 |

## Описание

Реализовать `FewShotRetriever` в файле `pipeline/retriever.py`. По тексту входящего сообщения ищет в Qdrant top-K семантически похожих пар «входящее → ответ пользователя», которые были проиндексированы на этапе E1.

Ключевое поведение: если максимальный `score` среди найденных примеров ниже порога `MIN_SCORE = 0.65`, значит у системы недостаточно похожих примеров, чтобы сформировать уверенный ответ. В этом случае `low_confidence=True`, и вместо черновика владелец получит уведомление «не могу ответить уверенно».

## Что нужно разработать

- Датакласс `RetrievalResult(examples: list[dict], low_confidence: bool)`
- Датакласс `FewShotExample(incoming: str, reply: str, score: float)`
- Класс `FewShotRetriever` с методом `retrieve(query: str, top_k: int = 5) -> RetrievalResult`
- Инициализация: подключение к Qdrant, загрузка модели `sentence-transformers`

## Критерий приёмки

- `retrieve()` возвращает `RetrievalResult` с полем `examples` — список `FewShotExample`
- Векторизация через `sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2`
- При `max(score) < 0.65` → `RetrievalResult(examples=[], low_confidence=True)`
- При `low_confidence=True` pipeline не пытается генерировать черновик
- Время поиска < 300 мс на коллекции до 10 000 пар
- Подключение к Qdrant конфигурируется через переменные окружения (`QDRANT_HOST`, `QDRANT_PORT`)

## Примерное решение

```python
# pipeline/retriever.py
import os
from dataclasses import dataclass, field
from qdrant_client import QdrantClient
from sentence_transformers import SentenceTransformer

MIN_SCORE = 0.65
COLLECTION = "fewshot_pairs"
MODEL_NAME = "sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2"


@dataclass
class FewShotExample:
    incoming: str
    reply: str
    score: float


@dataclass
class RetrievalResult:
    examples: list[FewShotExample] = field(default_factory=list)
    low_confidence: bool = False


class FewShotRetriever:
    def __init__(self):
        self._model = SentenceTransformer(MODEL_NAME)
        self._client = QdrantClient(
            host=os.getenv("QDRANT_HOST", "localhost"),
            port=int(os.getenv("QDRANT_PORT", 6333)),
        )

    def retrieve(self, query: str, top_k: int = 5) -> RetrievalResult:
        vector = self._model.encode(query).tolist()

        hits = self._client.search(
            collection_name=COLLECTION,
            query_vector=vector,
            limit=top_k,
        )

        if not hits or hits[0].score < MIN_SCORE:
            return RetrievalResult(examples=[], low_confidence=True)

        examples = [
            FewShotExample(
                incoming=hit.payload["incoming"],
                reply=hit.payload["reply"],
                score=hit.score,
            )
            for hit in hits
        ]
        return RetrievalResult(examples=examples, low_confidence=False)
```
